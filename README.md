# SQL-Business-Performance-Analysis
SQL Business Performance Analysis using SQLite | Data Analytics Portfolio Project
Overview

This project analyzes Walmart sales data using SQL to generate actionable business insights. The analysis covers sales performance, customer behavior, branch performance, product categories, payment methods, and revenue trends using SQLite.

Objectives
Perform exploratory data analysis using SQL
Analyze sales performance across branches and cities
Identify top-performing product categories
Evaluate customer ratings and payment methods
Generate business insights using SQL queries

Dataset
The dataset contains Walmart sales transaction records, including:
Invoice ID
Branch
City
Category
Unit Price
Quantity
Date
Time
Payment Method
Customer Rating
Profit Margin

Business Insights

Compared sales performance across different product categories.
Analyzed customer satisfaction through ratings.
Evaluated payment method preferences.
Tracked monthly revenue trends to identify sales patterns.
Generated executive-level KPIs for business decision-making.
